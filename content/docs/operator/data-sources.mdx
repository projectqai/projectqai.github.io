## Opening the Configuration Menu

Press `Ctrl+K` (or `⌘+K` on macOS) to open the command menu, then select **Configuration**.

## Enabling ADS-B

Let's use the builtin mission controller "Playground" to quickly enable some public data feeds.

1. In the configuration menu, open **Missions → Playground**
2. Toggle **Enable ADS-B** on
3. Click **Save**
4. Close the menu by pressing `Esc`, clicking **Back**, or clicking outside the menu

After a moment, aircraft entities will begin appearing on the map. Clicking any of them will open them in the entity details panel.
