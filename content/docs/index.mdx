## Installing

<Tabs items={['Windows Desktop', 'Mac Desktop', 'macOS', 'Windows', 'Linux', 'Android', 'Docker', 'Kubernetes']}>
  <Tab value="Windows Desktop">
    Download the latest [Windows desktop release](https://github.com/projectqai/hydris/releases/download/v0.0.20/hydris-desktop-windows-amd64-v0.0.20.zip), extract it, and run the application.
  </Tab>
  <Tab value="Mac Desktop">
    Download the latest [Mac desktop release](https://github.com/projectqai/hydris/releases/download/v0.0.20/hydris-desktop-macos-arm64-v0.0.20.zip) and extract it.

    macOS will say that Apple cannot verify the app. To allow it:

    1. Open **System Settings** → **Privacy & Security**
    2. Scroll down to the message saying the app "was blocked to protect your Mac"
    3. Click **Open Anyway**
  </Tab>
  <Tab value="macOS">
    Download the [macOS binary](https://github.com/projectqai/hydris/releases/latest/) and run:

    ```bash
    chmod +x hydris
    ./hydris
    ```

    If macOS blocks the unsigned binary, remove the quarantine attribute:

    ```bash
    xattr -d com.apple.quarantine hydris
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Windows">
    Download the [Windows executable](https://github.com/projectqai/hydris/releases/latest/) and run:

    ```bash
    ./hydris.exe
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Linux">
    Download the [Linux binary](https://github.com/projectqai/hydris/releases/latest/) (x86_64 or ARM):

    ```bash
    chmod +x hydris
    ./hydris
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Android">
    1. On your Android device, open [the latest release](https://github.com/projectqai/hydris/releases/latest/) in your browser
    2. Download the `.apk` file
    3. When prompted, allow installation from unknown sources
    4. Tap the downloaded APK to install
  </Tab>
  <Tab value="Docker">
    Run Hydris using Docker:

    ```bash
    docker run -p 50051:50051 -ti ghcr.io/projectqai/hydris:latest
    ```

    To persist world state across restarts, add a volume:

    ```bash
    docker run -p 50051:50051 -v hydris-data:/data -ti ghcr.io/projectqai/hydris:latest -w /data/world.yaml
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Kubernetes">
    Deploy Hydris to your cluster using the manifest from the repo:

    ```bash
    kubectl apply -f https://raw.githubusercontent.com/projectqai/hydris/main/scripts/deploy.yaml
    ```
  </Tab>
</Tabs>
