## Quick Start

### 1. Download and Run Hydris

<Tabs items={['macOS', 'Windows', 'Raspberry Pi', 'Android', 'Docker']}>
  <Tab value="macOS">
    Download the [macOS binary](https://github.com/projectqai/hydris/releases/latest/) and run:

    ```bash
    chmod +x hydris
    ./hydris
    ```

    If macOS blocks the unsigned binary, remove the quarantine attribute:

    ```bash
    xattr -d com.apple.quarantine hydris
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Windows">
    Download the [Windows executable](https://github.com/projectqai/hydris/releases/latest/) and run:

    ```bash
    ./hydris.exe
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Raspberry Pi">
    Download the [ARM Linux binary](https://github.com/projectqai/hydris/releases/latest/):

    ```bash
    chmod +x hydris
    ./hydris
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
  <Tab value="Android">
    Download the [APK](https://github.com/projectqai/hydris/releases/latest/) and install on your device.

    **Sideloading with ADB:**

    1. Enable Developer Options on your Android device:
       - Go to Settings → About Phone
       - Tap "Build Number" 7 times

    2. Enable USB Debugging:
       - Go to Settings → Developer Options
       - Enable "USB Debugging"

    3. Connect your device and install:
       ```bash
       adb install -r hydris.apk
       ```
  </Tab>
  <Tab value="Docker">
    Run Hydris using Docker:

    ```bash
    docker run -p 50051:50051 -ti ghcr.io/projectqai/hydris:v0.0.14
    ```

    Hydris starts on `localhost:50051` with a web UI at `http://localhost:50051`
  </Tab>
</Tabs>

### 2. Connect Your First Client

Install the client libraries for your language:

<Tabs items={['TypeScript', 'Python', 'Go']}>
  <Tab value="TypeScript">
    ```bash
    npm install @projectqai/proto @bufbuild/protobuf @connectrpc/connect @connectrpc/connect-web
    ```
  </Tab>
  <Tab value="Python">
    ```bash
    pip install grpcio grpcio-tools
    ```
  </Tab>
  <Tab value="Go">
    ```bash
    go get google.golang.org/grpc
    go get github.com/projectqai/proto/go
    ```
  </Tab>
</Tabs>

### 3. Create Your First Entity

<Tabs items={['TypeScript', 'Python', 'Go']}>
  <Tab value="TypeScript">
    ```typescript
    import { EntitySchema, WorldService } from "@projectqai/proto/world";
    import { create } from "@bufbuild/protobuf";
    import { createClient } from "@connectrpc/connect";
    import { createConnectTransport } from "@connectrpc/connect-web";

    // Create an entity
    const entity = create(EntitySchema, {
      id: "sensor-1",
      geo: {
        latitude: 52.52,
        longitude: 13.40,
        altitude: 100
      },
      symbol: {
        milStd2525C: "SFGPES----"
      }
    });

    // Connect to Hydris and push the entity
    const transport = createConnectTransport({
      baseUrl: "http://localhost:50051"
    });
    const client = createClient(WorldService, transport);

    const response = await client.push({
      changes: [entity]
    });
    ```
  </Tab>
  <Tab value="Python">
    ```python
    from world_pb2 import Entity, GeoSpatialComponent, SymbolComponent

    entity = Entity(
        id='sensor-1',
        geo=GeoSpatialComponent(
            latitude=52.52,
            longitude=13.40,
            altitude=100
        ),
        symbol=SymbolComponent(
            milStd2525C='SFGPES----'
        )
    )
    ```
  </Tab>
  <Tab value="Go">
    ```go
    import pb "github.com/projectqai/proto/go"

    entity := &pb.Entity{
        Id: "sensor-1",
        Geo: &pb.GeoSpatialComponent{
            Latitude:  52.52,
            Longitude: 13.40,
            Altitude:  100,
        },
        Symbol: &pb.SymbolComponent{
            MilStd2525C: "SFGPES----",
        },
    }
    ```
  </Tab>
</Tabs>
