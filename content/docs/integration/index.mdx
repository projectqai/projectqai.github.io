---
title: Builtin Integrations
description: Connecting to Hydris from external systems
---

Hydris can be integrated from any language using the gRPC/HTTP API, or connected to existing tactical and surveillance systems using built-in protocol adapters that require no code.

If you haven't installed Hydris yet, follow the [installation instructions](/docs#installing) first.


## Non-Code Integrations

If your system supports standard protocols, no code is required to connect them to Hydris.
Integrations are configured via the command menu -> Configuration

| Protocol | Use Case |
|----------|----------|
| [ATAK / TAK](/docs/integration/atak) | Cursor-on-Target (also known as TAK) |
| [ASTERIX](/docs/integration/asterix) | Surveillance radar track data |
| [SAPIENT](/docs/integration/sapient) | The standard for Counter UAS |
| [MAVLink](/docs/integration/mavlink) | Drone and unmanned vehicle telemetry |
| [AIS](/docs/integration/ais) | Maritime vessel tracking |
| [SpaceTrack](/docs/integration/spacetrack) | Satellite tracking via TLE/SGP4 |
| [Reolink](/docs/integration/reolink) | IP cameras with RTSP streams and PTZ control |
